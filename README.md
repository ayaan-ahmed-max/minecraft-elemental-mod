# minecraft-elemental-mod
Elemental superpowers for Minecraft 26.2 that won't melt your potato PC. Forge mod, zero tick handlers, configurable particle budget.

**➡️ [Setup & How to Play](SETUP.md)** — installation, crafting recipes, abilities, and low-end tuning.

Requires Minecraft 26.2 + Forge 65.0.3. Build with `./gradlew build` (JDK 25); the jar lands in `build/libs/`.
